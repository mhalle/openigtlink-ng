Browser WS demo

Minimal example of @openigtlink/core/net/ws running in a browser. Connects to a Python or Node WS server speaking OpenIGTLink (one binary frame = one IGTL message), sends TRANSFORM messages, and logs STATUS replies.

How to serve the Python side

from oigtl.net import Server
from oigtl.messages import Transform, Status

server = await Server.listen_ws(18945)

@server.on(Transform)
async def _(env, peer):
    await peer.send(Status(code=1, status_message=f"got {env.body.matrix[-3:]}"))

await server.serve()
